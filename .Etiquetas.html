Titulo mas importante -> h1

Para poner una palabra/frase imporante (palabras clave) -> strong

Para poner en cursiva una palabra/frase -> em

Para subrayar una palabra/frase ->u

Para tachar una palabra/frase ->s

Para poner subindice H2O ->sub

Para poner elevado X3 ->sup

Para modificar los estilos del texto ->font

Para darle color, asi en rojo, font color=XXXXXX

Propiedades color: R-G-B --->FF0000 -> rojo; / 00FF00 -> verde / 0000FF -> azul

Para modificar tamaño pequeño, font size="1" hasta grande, font size="7"

Para modificar tipo de letra, font face="comic sans ms"


Salto de linera ->br/
Linea horizontal ->hr/

Admite las propiedades [color="green" (color de la linea)], [size="50px"(grosor de la linea)], [width="50%" (ancho de la linea)] y [align=left/right/center (donde se alinea la linea)].
